Static variable. Controls whether Appliance.matches() allows synonyms for appliance type names.

Static (AKA class) variable. Maps from appliance_type (string) to a dict describing metadata about each appliance type.

label(pretty=False)[source]¶

Return string ‘(<type>, <identifier>)’ e.g. ‘(fridge, 1)’ if pretty=False else if pretty=True then return a string like ‘Fridge’ or ‘Fridge 2’. If type == ‘unknown’ then appends original_name to end of label.

matches(key)[source]¶

Returns True if all key:value pairs in key match appliance.metadata or Appliance.appliance_types[appliance.metadata[‘type’]]. Returns True if key is empty dict.

By default, matches synonyms. Set Appliance.allow_synonyms = False if you do not want to allow synonyms.
